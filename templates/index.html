UW Course Tool

The UW Course Tool helps to simplify course searches at UW. It offers a keyword search to search through the descriptions of every course offered at UW to give you an idea of what courses could match your interests. Each department for every UW Campus is listed with a link to a page with all courses in that department. Prerequisite Trees help give you a better understanding of the courses you need to take and succint course information gives you a quick overview of every course. The search bar in the top right of the screen allows you to quickly search for courses and departments.


Requirements

The Requirements show you all the files/settings that need to be in place in order for course searches and prerequisite tree generation to work: A Transcript, Internet Connection and Course Files. Whether each of these requirements is fulfilled is indicated by a green checkmark above the item. If all required files are present, then the internet connection is not needed. The Transcript is optional, and upon successfull uploading, the Transcript will have a green checkmark above it. If you chose to scan in your transcript, all courses you've already taken will be removed from Prerequisite Trees and will not show up in the Keyword Search. This can make Prerequisite Trees for higher level courses much more readible and useful.
The UW Course Catalogs do get updated every once in a while, so if you would like to update the current course files to include the newest courses, you can click the 'Update Course Catalogs' button. Keep in mind that downloading all the information and creating all the necessary information can take up to 3 minutes, so update the catalogs sparingly.



Departments

Departments shows you all the Departments for each UW Campus. Clicking on the Department abbreviation will lead you to a page with a list of courses in that department. Clicking on any one of those courses will bring you to another page with all the information for that course including its Prerequisite Tree if there are any prerequisites for the course.



Create Tree

The Tree Generator creates a Prerequisite Tree for any course you enter. Courses must be the department abbreviation followed by the 3 digit course code. Example: EE235



Keyword Search

The Keyword Search shows you all courses with the given keyword in their description (case-insensitive).



MyMap

MyMap is a tool you can use to plan out your schedule, on a map! Enter all the courses you want to take and when you are done, click 'Create Schedule'. All possible combinations of courses without overlapping sections will be shown in the header and their location will be displayed with a pin on the map. You can select which UW Campus you want to create mapped out schedules for: Seattle, Bothell or Tacoma. Clicking on the pins on the map will give you information about the course section in that building.
